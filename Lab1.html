El propósito de GET es para solicitar datos de algun sitio, el de HEAD es para obtener solo encabezados de una respuesta, el de POST es para mandar data a algun serivdor y crear un recurso.
el de PUT es para editar, actualizar un recurso en un servidor y en el cuerpo esta el mensaje que se agregará, el de PATCH es para aplicar modificaciones parciales al recurso
y el de DELETE es para eliminar el recurso especificado.
